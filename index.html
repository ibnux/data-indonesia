Data Indonesia

Data Provinsi, kabupaten, kecamatan dan kelurahan di Indonesia dalam bentuk json

Source Code Demo

Struktur Data

  • provinsi.json
  • kabupaten/[id provinsi].json isinya data kabupaten sesuai id provinsi
  • kecamatan/[id kabupaten].json isinya data kecamatan sesuai id kabupaten
  • kelurahan/[id kecamatan].json isinya data kelurahan sesuai id kecamatan

Saat awal biasanya request dulu provinsi.json, ditambahkan ke autocomplete

Saat provinsi dipilih misal ACEH dengan id 11, lalu rekues ke folder kabupaten/kota untuk mendapatkan daftar kabupaten/kota. contoh kabupaten/11.json atau kota/11.json

Saat kabupaten dipilih misal KABUPATEN SIMEULUE dengan id 1101, lalu rekues ke folder kecamatan untuk mendapatkan daftar kecamatan. contoh kecamatan/1101.json

Saat kecamatan dipilih misal TEUPAH SELATAN dengan id 1101010, lalu rekues ke folder kelurahan untuk mendapatkan daftar kelurahan. contoh kelurahan/1101010.json

dan terpilihlah misal kelurahan LATIUNG dengan id 1101010001

Struktur ID 1101010001 = 11+01+010+001 = provinsi+kabupaten+kecamatan+kelurahan
